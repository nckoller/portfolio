Movie Hub (Group Project)

For my first group project we were tasked to create an app that used two different API calls. Using Open Movie DataBase (OMDB) and The Movie Database (TMDB), we created an app that will suggest three random movies based on various sets of criteria.

Screamcast (2nd Group Project)

For our second group project we were tasked with building a full-stack web application using the MVC paradigm and a server side API. We designed a scary movie database that would allow users to register accounts, leave written reviews, and "spooky" ratings for movies. The reviews are displayed alongside the movies and the "spooky" rating is updated as reviews are submitted.

Live App GitHub Repository

This application is a simple full-stack development CRUD application. The app allows the user to enter any burger name to add it to the menu, and the MySQL database. The user can then click the "EAT ME!" button and turn the "Devoured" status to "true", moving it to the "Devoured" menu.
